Informações sobre o Aplicativo DM Programação Tela Inteligente

Versão: 1.0

Descrição:
Este aplicativo foi desenvolvido para facilitar o acesso a diversas funcionalidades em um único lugar, incluindo:
- Reprodução de músicas e rádios online
- Acesso rápido a sites e ferramentas de trabalho
- Visualização de vídeos
- Dashboard de informações

Funcionalidades Principais:
1. Música: Permite buscar e reproduzir rádios online
Funcionalidades:
1. Web: Acesso rápido a sites úteis
2. Trabalho: Ferramentas para produtividade
3. Redes: Acesso a redes sociais
4. Vídeos: Player de vídeos
5. Dashboard: Visualização de dados

Tecnologias:
- Python 3.10+
- CustomTkinter
- VLC Player
- Windows Media Player
- API Radio-Browser
